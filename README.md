CNC Machine Monitoring System

A smart monitoring system designed to track CNC machine operations, collect machine data, and provide real-time insights into machine performance and status. The project aims to improve operational visibility, reduce downtime, and support efficient manufacturing processes.

Features:
Real-time monitoring of CNC machine parameters
Tracking of machine status and operational metrics
Alert generation for abnormal machine conditions
Dashboard/reporting for performance analysis
Data collection and storage for historical analysis
Scalable design for industrial IoT applications

How It Works:
Data Collection: The system receives operational data from CNC machines or connected sensors.
Data Processing: Collected data is processed and stored in the database for real-time and historical analysis.
Monitoring & Alerts: The application continuously monitors machine parameters and generates alerts when abnormal conditions are detected.
Visualization: A dashboard displays machine status, performance metrics, and reports for operators and administrators.

Getting Started
Prerequisites:
Python 3.x installed
MySQL/MongoDB installed and running
Git installed

Installation:
Clone the repository
Install dependencies
Configure the database connection in the project settings.
Run the application
